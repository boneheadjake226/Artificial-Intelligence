To compile Hello_World.cpp
	1)Open terminal window
	2)Navigate to directory of Hello_World.cpp
	3)compilation command: $ g++ -o Hello_World Hello_World.cpp
To Run Hello_World.cpp
	1)Copy testing .txt files into same directory as previousely compiled Hello_World application
	2)run command: $ Hello_World
	3)type name of testing file in the given input field
